(4) Edit the file, add the above code [Don't worry, you don't need to modify anything unless if you want] and save the file,
(5) Open Command prompt (for windows users)/Terminal (for macOS and Linux users), go to the directory where you have saved the file, enter the commands `javac "Emoji_updates_of_this_discord_server.java"` and `java "Emoji_updates_of_this_discord_server"`
